指標（Pointer）是一種變數，用來儲存另一個變數的記憶體位址。

它不是直接存放數值，而是儲存該數值在記憶體中的位址。

兩個常用：

-   解參考運算子（\*）：用來宣告指標變數，或用來存取某個位址所儲存的值。
-   位址運算子（&）：用來取得變數的位址，或把某變數的位址傳給指標使用。

### 範例：印出地址

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int m = 100;
    int* ptr = &m;

    cout << "The Value of Variable m is: " << m << endl;
    cout << "The Memory Address of Variable m is: " << &m << endl;
    cout << "The Memory Address of Variable m is using ptr: " << ptr << endl;

    return 0;
}
```

{% endraw %}

### 範例：印出指標值與地址

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int* ptr;
    ptr = &a;

    cout << "ptr = " << ptr << endl;
    cout << "*ptr = " << *ptr << endl;

    return 0;
}
```

## 元素指標（Element Pointer）

元素指標是指向陣列第 0 個元素的地址的指標。

當你寫 `int *ptr = arr;`，代表 ptr 指向 `arr[0]`，也就是陣列的第 0 個元素。

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int* ptr = arr;  // ptr 指向 arr[0]

    cout << "Address of arr[0]: " << ptr << endl;

    return 0;
}
```

{% endraw %}

### 範例：（重要）存取陣列元素的位址

使用 `ptr + i`（指標運算）：

-   `ptr + i` 計算第 i 個元素的位址。
    使用 `&ptr[i]`（語法糖）:
-   `&ptr[i]` 直接取得第 i 個元素的位址。
-   這是` ptr + i` 的語法糖，編譯器會將其轉換為 `ptr + i`。

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int* ptr = arr; // 指向 arr[0]

    // 指標運算
    cout << "Pointer arithmetic (ptr + i):\n";
    for (int i = 0; i < 5; ++i) {
        cout << (ptr + i) << '\n';
    }

    // 語法糖：&ptr[i]
    cout << "\nAddress using &ptr[i]:\n";
    for (int i = 0; i < 5; ++i) {
        cout << &ptr[i] << '\n';
    }

    return 0;
}

```

{% endraw %}

### 範例：（重要）存取陣列元素的值

使用 `*(ptr + i)`（指針算術）
使用 `ptr[i]`（語法糖）

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int* ptr = arr;

    // 指標算術
    cout << "Values via *(ptr + i): ";
    for (int i = 0; i < 5; ++i) {
        cout << *(ptr + i) << "\t";
    }
    cout << "\n";

    // 語法糖 ptr[i]
    cout << "Values via ptr[i]:     ";
    for (int i = 0; i < 5; ++i) {
        cout << ptr[i] << "\t";
    }
    cout << "\n";

    return 0;
}
```

## 陣列指標（Array Pointer）

陣列指標是指向整個陣列的指標，不是指向單一元素。

最常用於 多維陣列工作 與 保留陣列大小資訊。

| 宣告方式              | 說明                       |
| --------------------- | -------------------------- |
| `int *p = arr;`       | 指向 `arr[0]`，第 0 個元素 |
| `int (*p)[5] = &arr;` | 指向整個 `arr[5]` 陣列     |

### 範例：區分指向元素與整個陣列

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int* p1 = arr;         // 指向 arr[0]
    int (*p2)[5] = &arr;   // 指向整個陣列 arr[5]

    // 存取元素
    cout << p1[2] << endl;        // 印出 30
    cout << (*p2)[2] << endl;     // 一樣印出 30

    // 指向資料的大小
    cout << "sizeof(*p1) = " << sizeof(*p1) << endl;  // 4（int）
    cout << "sizeof(*p2) = " << sizeof(*p2) << endl;  // 20（5 ints）

    // 指標本身的大小（通常是 8 bytes on 64-bit systems）
    cout << "sizeof(p1) = " << sizeof(p1) << endl;
    cout << "sizeof(p2) = " << sizeof(p2) << endl;

    return 0;
}

```

{% endraw %}

### 範例：指向單一元素 vs 整個陣列的指標行為

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int* p1;         // 指向單一整數（arr[0]）
    int (*p2)[5];    // 指向整個陣列（arr）

    p1 = arr;
    p2 = &arr;

    // 印出地址
    cout << "p1  = " << p1 << endl;
    cout << "*p2 = " << *p2 << "\n\n";

    // 向後移動
    p1++;  // 前進 1 個 int（4 bytes）
    p2++;  // 前進 1 個陣列（5 * sizeof(int) = 20 bytes）

    cout << "p1  = " << p1 << endl;
    cout << "p2  = " << p2 << endl;

    // 印出值
    cout << "p1[0] = " << *p1 << endl;       // arr[1]
    cout << "p1[1] = " << *(p1 + 1) << endl; // arr[2]

    // 注意：*(p2)[0] 指的是「p2 指向的陣列」的第 0 個元素
    cout << "p2[0][0] = " << (*p2)[0] << endl; // 可能是 arr[5] 範圍之外的值（不保證）

    return 0;
}

```

{% endraw %}

## 3. 指標操作二維陣列

```cpp
int arr[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

// 其實在記憶體中是一條直線
// [1][2][3][4][5][6]
```

### 範例：操作方式 01：用 int\* 線性存取整個二維陣列

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    int* p = &arr[0][0];  // 指向第一個元素

    for (int i = 0; i < 6; ++i) {
        cout << *(p + i) << " ";
        // 或使用語法糖：
        cout << p[i] << " ";
    }
    cout << endl;

    return 0;
}

```

{% endraw %}

### 範例：### 範例：操作方式 02：使用二維指標運算存取元素

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int num[2][4] = {
        {312, 16, 35, 65},
        {52, 111, 77, 80}
    };

    // 二維陣列本體即為指標（指向第一列）
    cout << "num         = " << num << endl;
    cout << "num + 1     = " << (num + 1) << endl;

    // 解參考方式取得列地址
    cout << "*(num)      = " << *(num) << endl;
    cout << "*(num + 1)  = " << *(num + 1) << endl;

    // 存取特定位置地址與值
    cout << "*(num) + 2         = " << *(num) + 2 << endl;
    cout << "*(num + 1) + 2     = " << *(num + 1) + 2 << endl;
    cout << "*(*(num) + 2)      = " << *(*(num) + 2) << endl;       // num[0][2]
    cout << "*(*(num + 1) + 2)  = " << *(*(num + 1) + 2) << endl;   // num[1][2]

    // 使用迴圈逐一印出所有元素
    size_t m = sizeof(num) / sizeof(num[0]);       // 列數
    size_t n = sizeof(num[0]) / sizeof(num[0][0]); // 行數

    for (size_t i = 0; i < m; ++i) {
        for (size_t j = 0; j < n; ++j) {
            cout << *(*(num + i) + j) << "\t"; // 或寫作 num[i][j]
        }
        cout << endl;
    }

    return 0;
}

```

## 函數指標（Function Pointer）

函數指標可以儲存「函數的位址」。

用於實現 callback（回呼）機制、行為抽象化，甚至模擬簡單 策略模式。

{% raw %}

```cpp
// 建立函數
int add(int a, int b) {
    return a + b;
}

// 宣告函數指標：指向返回 int 且接受兩個 int 的函數
int (*fptr)(int, int);

// 初始化函數指標
fptr = &add;   // 可寫作 fptr = add;

// 呼叫方式
int result1 = (*fptr)(3, 4); // 解參考方式
int result2 = fptr(3, 4);    // 直接呼叫方式

```

{% endraw %}

### 範例：基本函數指標範例

{% raw %}

```cpp
#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    int (*fptr)(int, int);  // 宣告指向函數的指標
    fptr = add;             // 初始化（可省略 &）

    cout << "Result: " << fptr(5, 3) << endl;  // 輸出: 8
    return 0;
```

```cpp
int* firstPositive(int* ptr) {

    return ptr;

    cout << "*iptr = " << *iptr << endl;

    return 0;

// *iptr = 5
```

### 範例：使用雙指標動態分配二維陣列（手動配置）

{% raw %}

```cpp
#include <iostream>
using namespace std;

int main() {
    int m = 2, n = 3;

    // 分配 m 行
    int** arr = new int*[m];

    // 每行分配 n 個整數
    for (int i = 0; i < m; ++i) {
        arr[i] = new int[n];
    }

    // 初始化陣列
    for (int i = 0; i < m; ++i) {
        for (int j = 0; j < n; ++j) {
            arr[i][j] = i * n + j + 1;
        }
    }

    // 輸出陣列
    for (int i = 0; i < m; ++i) {
        for (int j = 0; j < n; ++j) {
            cout << arr[i][j] << " ";
        }
        cout << endl;
    }

    // 釋放記憶體
    for (int i = 0; i < m; ++i) {
        delete[] arr[i];
    }
    delete[] arr;

    return 0;
}
```

## 空指標(Null Pointer)

在 C++ 中，空指標（Null Pointer） 表示「不指向任何有效記憶體位置」的指標。

-   C++11 之前使用 NULL（本質上是 0）
-   C++11 之後建議使用 nullptr（專門為指標設計的常數，型別安全）

常見用途

-   初始化指標，避免未定義行為
-   判斷記憶體分配是否成功
-   作為資料結構的結尾標記（如 linked list）

{% raw %}

```cpp
int* ptr = nullptr; // 建議用法（C++11+）

// 舊寫法（仍可用，但不推薦）
int* ptr2 = NULL;
int* ptr3 = 0;  // 可隱式轉為 nullptr
```

{% endraw %}

### 範例：初始化空指標並檢查

```cpp
#include <iostream>
using namespace std;

int main() {

        cout << "Pointer is NULL (nullptr)" << endl;
    } else {
        cout << "Pointer is NOT NULL" << endl;
    }

    return 0;
}
```

## NULL Pointer vs Void Pointer

| 特性     | Null Pointer（`nullptr`）       | Void Pointer（`void*`）                  |
| -------- | ------------------------------- | ---------------------------------------- |
| 定義     | 不指向任何有效位址的特殊值      | 一種「通用指標型別」，可指向任意型別資料 |
| 類型     | 所有指標型別皆可設為 `nullptr`  | 僅能宣告為 `void*`                       |
| 比較性質 | 所有空指標皆等於 `nullptr`      | 不同 `void*` 值不一定相等                |
| 範例     | `int* p = nullptr;`             | `void* p = &someVar;`                    |
| 轉換限制 | 不可直接對 `nullptr` 做算術運算 | `void*` 需轉型才能存取實值               |

## 總結：pointer 優點

-   可有效存取與操作記憶體
-   適用於 動態記憶體管理
-   可用於構建靈活的資料結構（如 Linked List、Tree、Graph）
-   提高函式傳遞彈性（如傳入大型陣列的位址）
-   可減少複製，提高效能與彈性

## 總結：pointer 缺點

| 風險類型      | 說明                                                      |
| ------------- | --------------------------------------------------------- |
| 記憶體洩漏    | 配置記憶體後未釋放，導致系統無法回收資源                  |
| 未初始化指標  | 指標未設為 `nullptr`，就操作會造成 **Segmentation Fault** |
| 指標算術錯誤  | 越界、錯誤的地址計算可能導致未定義行為                    |
| 多重 `delete` | 重複釋放記憶體會導致不穩定行為與程式崩潰                  |
