## 5. 每周 Lab

### 5.1. Lab01

-   註冊 leetcode 帳號
-   完成環境建置(完成 C 跟 C++的編譯建置)
    -   可以使用以下 2 種程式碼進行測試

```c
// TODO: C 的 Hello World
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

```

```c++
// TODO: C++ 的 Hello World
#include <iostream>

using namespace std;

int main() {
  cout << "Hello, World!" << endl;
  return 0;
}
```
